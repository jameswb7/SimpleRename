# SimpleRename

SimpleRename is a plugin designed to edit item names & lore with ease.

# Commands:
```
/rename <name>
/rename remove
/lore add <line;text> <text>
/lore set <line;text> <text>
/lore delete <line;text> <text>
/lore clear <line;text> <text>
```
# Permission Node:
```
command.rename
command.lore
```
